# IT-Service-Desk-Streamlit-Dashboard

IT Service Desk Dashboard built with Streamlit for interactive monitoring of service desk operations, featuring KPI tracking, ticket trend analysis, SLA insights, technician performance, problem analysis, and date-based filtering with data-driven visualizations.

🚀 Live App Demo: https://it-service-desk-app-dashboard.streamlit.app/

## Features
- Operational KPI Monitoring
- Ticket Trend Analysis
- SLA Compliance Insights
- Technician Performance Tracking
- Problem & Root Cause Analysis
- Interactive Date-Based Filtering
- Dynamic Data Visualizations

## Tech Stack
- Python
- Streamlit
- Pandas
- Plotly
